# マルチエージェント状態確認（Status）ワークフロー

このワークフローは、マルチエージェント・オーケストレーションの「Phase 2（監視）」を実行します。

1. **現在の状態読取**: `c:\AI\Agent\.agent\task.md` を読み込み、全サブタスクの現在の進捗を特定します。
2. **各エージェントの進捗確認**: ブロックされているエージェントがいないか `artifacts/questions/` を確認し、必要に応じて `read_terminal` で現在のログを調査します。
3. **ユーザーへの通知**: 進捗状況のサマリーを提示し、万が一エージェントの介入・再指示が必要な場合は警告を出します。
