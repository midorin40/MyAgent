# マルチエージェント計画（Plan）ワークフロー

このワークフローは、マルチエージェント・オーケストレーションの「Phase 0（計画）」を実行します。

1. **ユーザー指示の分析**: あなた（ユーザー）からの指示を受け取ります（例: `/plan アプリXについてのnote記事を作成して`）。
2. **能力マトリクスの参照**: `c:\AI\Agent\.agent\skills\multi-agent-browser-sync\SKILL.md` に定義されたエージェントの能力マトリクスを参照します。
3. **計画の立案**: 指示を直列または並列のサブタスクに分解します。
4. **タスク状態の作成**: テンプレートを使用し、各サブタスクに最適なエージェント（Claude, Gemini, Codex, Antigravity）を割り当てた `.agent/task.md` を生成します。
5. **計画の提示**: 生成した `.agent/task.md` をユーザーに提示し、確認を求めます。
